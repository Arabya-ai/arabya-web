# RAG + LLM (مرحلة D)

## الحالي

`/api/study` — استرجاع محلي فقط + ملخص موجز بلا نموذج لغوي.

## المستهدف

1. نفس الاسترجاع (آيات + كلمات + إعراب + تفسير)
2. طبقة اختيارية `mode=llm` عند ضبط `ARABYA_LLM_ENABLED=1` و`ARABYA_LLM_API_KEY`
3. الإجابة يجب أن تستشهد بـ `verseKey` / `wordId`
4. بدون المفتاح: يبقى السلوك الحالي (retrieval)

## الأمان

- لا تُرسل كتب محمية لطرف ثالث بلا موافقة الترخيص
- حد أقصى لطول السياق
- لا تخزين محادثات في Git
