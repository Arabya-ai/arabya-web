# ADR-0001 — المحتوى على Git أولاً

## الحالة

مقبول

## السياق

Arabya يحتاج تخزين نص القرآن والإعراب والتفاسير بتكلفة منخفضة وقابلية نسخ كاملة.

## القرار

كل المحتوى المرجعي شبه الثابت يُحفظ كـ JSON داخل الريبو تحت `/data` ويُنشر مع Vercel.

## النتائج

- مجاني دائمًا للمحتوى القرآني
- مراجعة عبر Pull Requests
- لا حاجة لقاعدة بيانات للمصحف
- الحسابات والملفات الكبيرة تُؤجَّل إلى D1/R2 عند الحاجة
