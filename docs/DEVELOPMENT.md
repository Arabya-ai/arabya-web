# عربية — سجل التطوير المؤجّل

آخر تحديث: 2026-07-23

## مؤجّل حتى اكتمال الأساس

### البلاغة
- إعادة طبقة «بلاغة» في `WordStudyDock` عند توفر مصادر مرخّصة.
- لا scraping لمواقع منافسة ولا التفاف على حقوق الغير.

### كتب الإعراب
- مسار `/books` يعرض الفهرس (معظم المداخل `awaiting_license`).
- `import-irab-book` + `import-from-incoming` + قناة IA المحلية — انظر `docs/platform/internet-archive.md`.
- مبدّل مصدر الإعراب في الدوك يُفعَّل فقط عند وجود مصدر `ready` إضافي (لم يُفعَّل بعد).

### الأحاديث والتراث
- صفحات `/hadith` و`/heritage` placeholders — لا تُعرض في التنقل حتى اكتمال محرّك القرآن.

### الموارد / القراءات الإضافية
- ورش، قالون، شعبة، مصحف التجويد الملون — بعد بيانات مرخّصة.
- صفحة `/resources` تبقى للروابط العامة عند الحاجة.

### الاشتراكات المدفوعة
- مؤجّلة بالكامل حتى اكتمال المنتج الأساسي (الطبقات، الحسابات، الجودة).
- لا تبنِ بوابة دفع الآن.

### Claims متعددة (ADR-0002)
- عند اختلاف آراء نحوية/دلالية من مصادر متعددة — بلا لوحة مصادر في الواجهة حاليًا.

## تم في هذه المرحلة
- طبقات الدراسة: إعراب (افتراضي) · صرف ومعجم · ترجمة ودلالة · تفسير — مع تلميحات وظيفية (بدون لوحة مصادر)
- تلميح يفصل أوضاع عرض الصفحة عن طبقات دراسة الكلمة
- توسيع معانٍ دراسية أصيلة (`lemma-sense-ar` / `build-meaning-ar`)
- تحسين سرد الإعراب وتسميات صرف إضافية
- قناة Internet Archive محلية (`fetch-ia-item`) + `incoming/` في gitignore
- فهرس `/books` يعرض الحالة دون تفعيل كتب بانتظار الترخيص
- إخفاء البلاغة من الواجهة
- تنظيف الفوتر والهيدر
- إصلاح تلاوة عبدالباسط والسديس (مجلدات EveryAyah 192kbps)
- شريط عنوان السورة بعرض كامل؛ بدون زخرفة على «مصحف المدينة»
- قصّ payload الإعراب لآيات صفحة المصحف فقط (أداء)
- تشديد مدخلات `/api/search` و`/api/study` + ترويسات أمنية/CSP
- اقتراح جذر صرفي من بحث الفهرس → `/root/[root]`
- Dependabot أسبوعي لـ npm / شهري لـ Actions
- مواءمة README مع الطبقات الأربع + روابط اكتشاف `/ayah` و`/root` من المصحف واللوحة
- اختبارات Vitest إضافية (قصّ الإعراب، صرف، سرد إعراب)
