# Arabya Web

موقع **تفسير كلمات القرآن الكريم** — واجهة عربية (RTL) على Next.js مع إعراب وتفاسير متعددة.

## التقنية

- **Next.js 15 + TypeScript + Tailwind 4**
- **البيانات:** ملفات JSON داخل `/data`
- **النشر المستهدف:** Vercel → `arabyaai.com` ثم `arabya.ai`

## مصادر البيانات (مفتوحة)

| المصدر | الاستخدام |
|--------|-----------|
| Quran.com API | نص QPC Hafs، ترجمات الكلمات، التفاسير |
| [Quranic Arabic Corpus](http://corpus.quran.com) | إعراب/صرف لكل كلمة (GPL — مع ذكر المصدر) |

## التشغيل محليًا

```bash
npm install
npm run fetch-data           # كلمات السور (QPC Hafs)
npm run apply-qpc-text       # إعادة تطبيق ترميز QPC إن لزم
npm run build-mushaf-index   # فهرس صفحات المصحف (604)
npm run build-irab           # بناء الإعراب من ملف الصرف
npm run fetch-tafsirs        # تحميل التفاسير العربية
npm run dev
```

## المسارات

- `/` — فهرس السور + متابعة آخر صفحة
- `/mushaf/[page]` — مصحف المدينة (1–604) مع دراسة الكلمات
- `/surah/[id]` — إعادة توجيه لأول صفحة مصحف للسورة
- `/about` · `/privacy`

## واجهة الدراسة

- **الكلمات** — جدول كلمة / ترجمة
- **الإعراب** — تحليل صرفي عربي لكل كلمة
- **تفسير السعدي / الميسر / ابن كثير** (يُحمَّل عند اختيار التبويب)
- تكبير/تصغير خط المصحف · نسخ رابط الآية · حفظ آخر صفحة قراءة
