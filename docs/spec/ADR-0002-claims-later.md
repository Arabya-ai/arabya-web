# ADR-0002 — Claims للمعاني والآراء لاحقًا

## الحالة

مقبول (مؤجّل التنفيذ الكامل)

## السياق

المعاني والتفاسير والآراء النحوية قد تختلف بين مصادر. نموذج Claim يسمح بربط كل قول بمصدر ودليل.

## القرار

- المرحلة الحالية: عرض مصدر واحد واضح لكل طبقة (Corpus للإعراب، Quran.com للترجمات).
- لاحقًا: تمثيل المعاني/الآراء كـ Claims مع `source` و`evidence` و`confidence`.
- لا نخلط الادعاءات المختلفة في حقل واحد بلا نسب.

## النتائج

- شفافية علمية أعلى عند التوسع
- توافق مع رؤية Knowledge Universe دون بناء Graph الآن
